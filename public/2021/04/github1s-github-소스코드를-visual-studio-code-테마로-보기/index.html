github1s - GitHub 소스코드를 Visual Studio Code 테마로 보기

April 15, 2021 in GitHub

공부를 하거나 연구를 하면서 내가 짠 코드가 아닌 다른 GitHub에 올라와있는 코드들을 보게 되는 경우가 생기는데요. 이 코드들을 더 예쁘게(?) 볼 수 있는 방법이 있어 소개하려 합니다. 바로 github1s입니다!!


github1s

본 포스팅에서는 제가 자주 뜯어보고 있는(?) functionaldata/tPACE repository를 사용해보겠습니다.

보통 GitHub에서 소스코드를 볼 때, 이렇게 해당 repository에 들어가서 보게 되는데요.

여기서, url의 github 부분을 github1s로 바꾸게 되면 놀라운 일이 생깁니다.

바로, Visual Studio Code처럼 소스코드들을 볼 수 있게 되는 것이죠!!!

또한 코드를 분할해서 볼 수도 있어 다른 사람들의 코드를 뜯어볼 때, 굉장히 유용합니다!!
