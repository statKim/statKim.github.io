blogdown으로 Github Blog 만들기

June 30, 2020 in blog

blogdown이란 Jekyll과 더불어 최근 인기가 많아지고 있는 Hugo 엔진을 사용하여 정적 페이지를 만들어주는 R 패키지입니다. 특히, R을 자주 사용하는 사람들은 Rstudio에서 코딩하고 Rmarkdown을 많이 사용하게 되는데, blogdown은 Rmarkdown 파일을 html로 변환시켜 페이지에 띄워주는 역할을 해줍니다. (R을 자주 사용하는 사람으로서 매우 편리하다고 생각!!)


순서

  1. R project 생성
  2. blogdown 설치
  3. Hugo 설치
  4. Site 생성
  5. Github에 commit


Rstudio를 켜고 R project를 만들어줍니다. 저는 blogdown_test라는 이름의 project를 생성했습니다. 생성된 후에는 폴더 안에 아무 파일도 없을겁니다. (이제 이것저것 생길거에요 ㅎㅎ)

이제 blogdown을 설치합시다.

install.packages("blogdown")
library(blogdown)

blogdown은 Hugo 기반이기 때문에 Hugo를 설치해주어야 합니다. blogdown에서는 Hugo를 설치해주는 함수가 있습니다. (매우 편리)

install_hugo()   # Hugo 다운로드

기본 theme을 사용해서 테스트해보겠습니다.

new_site()   # site 생성

기본 theme을 다운받으면서 자동으로 build 및 호스팅해주며 깔끔한 페이지가 나옵니다.

blogdown_test 폴더를 보면 파일이 많이 생겼죠? 이중에 실제로 호스팅은 public/ 폴더가 해줍니다. 나머지 파일을 사용해서 페이지를 build해주고 build된 파일들이 public/ 폴더에 들어가게 되요.

이 글에서는 기본 theme을 사용했지만 사실 Hugo에는 아주 많은 theme이 있습니다. (여기에서 맘에 드는 것을 찾으면 됩니다.)
